Это мульти-агентная система использует гигачат для запуска нужно создать файл config.py и положить туда ключ от LLM.

Эта система состоит из 2 агентов - Интервьюера и Обозревателя.

Интервьюер задаёт вопросы кандидату и отправляет ответы Обозревателю. Обозреватель их оценивает и на их основании генерирует новый вопрос.
Интервью продолжается до тех пор, пока кандидат не напишет в ответ стоп слово.
После этого выдаётся фидбэк и логи работы мультиагентной системы в файле interview_log.json
